## installation

make sure to backup your current config, then clone this repo to `~/.config/nvim`:

```bash
git clone https://github.com/aanes1/nvim ~/.config/nvim
```

## ghostty config

```toml
background = "191719"
macos-titlebar-proxy-icon = "hidden"
title = ""
font-family = "Iosevka Nerd Font"
font-size = 16
```
